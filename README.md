A modern, fully responsive, and professional developer portfolio built using **Flask**, **HTML**, **CSS**, and **JavaScript**. This project was developed as part of my internship at Thiranex, showcasing my projects, technical skills, certifications, and achievements in an elegant and interactive way.

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/bhargavasaiii17-svg/Thiranex_Portfolio.git
```

Go to project folder

```bash
cd Thiranex_Portfolio
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run Flask Application

```bash
python app.py
```

Open Browser

```
http://127.0.0.1:5000
```
